# ConvertSphere

AI Powered Universal Document Conversion SaaS.

## Features

- DOCX to PDF
- PPTX to PDF
- XLSX to PDF
- OCR Support
- AI Extraction
- Redis Queue Workers
- Real-time Updates
- Stripe Billing
- Clerk Authentication
- S3 Storage
- Dockerized Infrastructure
